# echo-lang
A custom shell language made for educational purposes only.
# How to use
## set var
```
test:=1
```
### call variable
```
test:=exit
test2:=hello world
print test2$
test$
```
The code above when typed in will print "hello world" then exit the program.
## print
```
print test
```
## exit
```
exit
```
